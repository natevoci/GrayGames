# Animal Catcher - Quick Start Guide

A gentle, colorful game designed for 4-year-olds to catch animated animals with a net.

## How to Play

1. Open `index.html` in a web browser
2. Move your mouse to control the butterfly net
3. Click your mouse or press **Space** to swoop and catch animals
4. Catch 20 animals to advance to the next level
5. Each level features a different animal with unique movement patterns

## Features

- **9 Levels** - Mice, Butterflies, Cats, Dogs, Fish, Crabs, Fairies, Gnomes, and Frogs
- **Bright, Friendly Design** - Colorful gradient background with rounded shapes
- **No Fail States** - Calm, pressure-free gameplay
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Smooth Animations** - Wobbling butterflies and fish for added charm
- **Score System** - Track points and progress

## Game Controls

| Action | Control |
|--------|---------|
| Move Net | Mouse or Touch |
| Catch Animal | Click or Space Bar |

## Animal Behaviors

Each animal has unique characteristics:
- **Speed variations** - Some animals move faster than others
- **Wobble effects** - Butterflies and fish wobble as they move
- **Different point values** - Harder animals earn more points

## Browser Compatibility

Works on all modern browsers:
- Chrome, Firefox, Safari, Edge
- Mobile browsers (iOS Safari, Chrome Mobile)

## Files

- `index.html` - Game structure and HTML
- `style.css` - Responsive styling
- `game.js` - Game logic and canvas drawing
