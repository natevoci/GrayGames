# Letter Hero - Quick Start Guide

## What is Letter Hero?
A fun, Guitar Hero-style typing game designed for 6-year-olds learning touch typing. Letters fall from the top, and kids press the matching keys to make them disappear and score points!

## How to Play

1. **Open** `index.html` in any web browser
2. **Press Start** to begin the game
3. **Watch** letters fall from the top of the screen
4. **Press the matching key** when a letter reaches the bottom
5. **Gain points** for accurate keypresses
6. **Level up** automatically as you scoreâ€”new letters unlock at each level!

## Game Features

### Controls (Top Bar)
- **Score**: Your current points
- **Level**: Current difficulty (1-5)
- **Speed Slider**: Adjust game speed (1 = slowest, 10 = fastest)
- **Start**: Begin a new game
- **Pause**: Pause/resume gameplay
- **Reset**: Clear score and restart

### Keyboard Display (Bottom)
The keyboard shows:
- **Colored keys** by finger assignment (left pinky, left ring, etc.)
- **Bright/active keys**: Letters you can type right now
- **Greyed-out keys**: Unlock as you level up

### Difficulty Progression
- **Level 1**: Home row only (A S D F J K L ;)
- **Level 2**: Add E and I
- **Level 3**: Add more top row letters
- **Level 4**: Top row + home row combined
- **Level 5**: All common keys

Advance levels by reaching 100 points per level!

## Tips for Kids
- Place your fingers on the home row (ASDFJ KL;) as shown on the keyboard
- Watch the keyboard colors to know which finger to use
- Start slow and gradually increase speed as you get faster
- Have funâ€”the game is forgiving!

## Browser Compatibility
Works on all modern browsers (Chrome, Firefox, Safari, Edge).
