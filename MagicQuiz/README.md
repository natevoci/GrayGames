## Quick Start

1. Open `index.html` in a web browser
2. Click "Let's Play!" to start
3. Read the question and type your answer
4. Press Enter to submit
5. Get 5 correct answers to level up!

## How It Works

- **Level 1-2**: Simple, shorter word answers
- **Level 3+**: More complex answers and longer words
- **Hints**: Get helpful hints if you're stuck
- **Level Tracking**: See your current level and progress to next level
- **No Scrollbars**: Game fits perfectly on screen

## Controls

- **Start Game**: Begin a new game
- **Pause**: Pause/resume gameplay
- **Reset**: Start completely fresh

## Customization

To add more questions, edit the `questionDatabase` in `game.js` and add questions to the appropriate level arrays. Each question needs:
- `question`: The question text
- `answer`: The correct answer (lowercase)
- `hint`: A helpful hint
- `illustration`: An emoji for visual appeal
